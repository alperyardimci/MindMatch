## English (U.S.) Localization

### Description
Think different, score points! MindMatch is a real-time multiplayer emoji telepathy game where you try to pick the emoji nobody else picks.

Each round, all players see the same set of emojis and secretly make their choice. When the picks are revealed, only players with unique selections score a point. If someone else picked the same emoji as you — neither of you scores!

GAME MODES:
• Random Play — Jump into a 5-player game instantly
• Create Room — Host a private game, set your own target score, and share the room code
• Join Room — Enter a friend's 4-character code to join their game

DUO TELEPATHY MODE:
When 2 friends create a room, the game flips! Now you want to pick the SAME emoji. Play 10 rounds and discover your telepathy score (0-100%) with a personalized fortune reading.

FEATURES:
• 500+ Twemoji emojis, randomized every round
• Real-time multiplayer via WebSocket
• Circular countdown timer with 15 seconds per round
• Emoji-grouped round results showing who connected
• Beautiful dark-themed UI with smooth animations
• Turkish and English language support

No ads. No tracking. No account required. Just pure telepathy fun.

### Notes for Review
MindMatch is a real-time multiplayer emoji game. To test the full experience:

1. **Random Play**: Tap "Random Play", enter any name, and confirm. You'll be matched with 4 AI opponents. Pick an emoji that you think nobody else will pick. Unique picks score points. First to 5 wins.

2. **Create Room**: Tap "Create Room" to get a room code. To test multiplayer, you'll need a second device or simulator with the app installed. The second player taps "Join Room" and enters the room code.

3. **Duo Telepathy Mode**: When exactly 2 human players are in a room, the game switches to Telepathy Mode — players try to pick the SAME emoji. After 10 rounds, a telepathy score and fortune reading is shown.

The app requires an active internet connection for gameplay as it uses WebSocket (Socket.io) for real-time communication with the game server.

No login required. No in-app purchases. No ads.
